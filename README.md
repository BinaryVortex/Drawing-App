# Drawing App

A simple, lightweight drawing application built with HTML, CSS, and JavaScript.

![Drawing App Screenshot](./Screenshot%202024-07-02%20144544.png)

## Features

- Freehand drawing with mouse or touch
- Selectable brush color
- Adjustable brush size
- Clear the canvas
- Save your drawing as an image

## Demo / Usage

1. Clone or download this repository.

   git clone https://github.com/BinaryVortex/Drawing-App.git

2. Open `index.html` in your web browser (no server required).

3. Controls
   - Use the color picker to change brush color.
   - Use the size slider to change brush thickness.
   - Click Clear to reset the canvas.
   - Click Save to download your drawing as a PNG.

## Files

- `index.html` — app markup
- `style.css` — styles
- `script.js` — drawing logic and interactions
